faq

why this site?
i built this mini game because i believe we need to strengthen and continually test our ability to detect AI-generated images (particularly those depicting humans) as AI content becomes more widespread online.

this is too easy to spot. can you make it harder?
i get it! i'm starting out with ~800 total images. though StyleGAN2 has strong performance + misses some of the more obvious AI 'tells', image generation is constantly improving. i'm working on adding more data + updating/playing around with models used to generate these images.

these are tough! how can i tell the difference?
AI image generation has gotten really good! i've compiled some resources that i'll eventually link here, but for now, practicing just like this helps improve your AI-intuition muscles.
